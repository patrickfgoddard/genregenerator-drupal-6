
-- SUMMARY --

The Genre Generator module is a simple, fun module for users to display a randomly generated module name.  The algorithm randomly grabs four descriptors and a top level genre (rock, pop, jazz, etc.). 

Module offers a page and block. ("/genre").

This is the original Drupal 6 version.


-- REQUIREMENTS --

None.

-- INSTALLATION --

Install as you would any other drupal module, either by drush or Drupal UI on modules admin page.
